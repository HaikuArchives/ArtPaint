The other windows

Besides the canvas of the main window, there are a number of other windows, you may invoke when needed, or decide to keep open to always be available at a glance. Do see the Keep in front options of the Settings to make some or all these windows float above the canvas.

ALT TAB will hide/show all these windows for an unobscured view of the canvas.

Colors

You can choose a different Color model from the menu: RGB, CMYK, HSV, CIELAB or BeOS. The additional "A" slider is the alpha channel, aka transparency.

When in RGB mode, holding SHIFT when clicking a slider sets the other sliders to that same value. It's a quick way to set a shade of gray.

Selecting a color with the left mouse button sets the foreground color, the right sets the background color. The same happens when you move a slider with the left/right mouse button.

Below the pane on the right, the color is shown as a hex-value that you can copy and paste when editing an HTML document, for example. The other way around is also possible, paste a hex-value from your HTML document to use the color in ArtPaint.

Under Color set you can create a New color set with 8 to 64 colors, Delete current set and Open/save color set.

The little arrow widgets beside the palette let you switch between the available color sets.

Layers

Here you see all layers stacked up on each other, with a checkbox that controls if a layer is visible.

The Layer menu has commands to Add a new layer and to Delete, Duplicate or Merge down (i.e. merge with one below) the currently selected layer.

You can arrange layers by dragging them around and rename them by editing their text field.

At the top is a slider to set a layer's transparency (Alpha) and a menu to set a blending Mode. Since ArtPaint implements these modes similarly to other graphics applications, you can find in-depth explanations for them online, e.g. in the Photoshop help or the GIMP docs.

Tools & Tool setup

Here you find all the available tools. By changing the Tools window size, you can re-arrange the icons.

A right-click opens the tool's setup window.

All tools and their settings are described on the pages of The tools.

Brushes

On first launch, the Brushes window is empty. You need to select the brush tool's setup window and configure it (size, hardness etc.), and then to store the brush.

Once you have configured a few different brushes - round, rectangular, big, small, soft, hard - you can quickly switch between them in the Brushes window.

The red line under the brush represents its scale: full-length means the representation is 1:1, half-length is 1:2 and so forth.

To remove a stored brush you can choose to Delete selected brush from the Brush menu.
