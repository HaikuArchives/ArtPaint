The Airbrush tool - A

This tool emulates a real-world airbrush.
You can set the Size of the airbrush and its Flow, i.e. how much paint is sprayed how quickly.

The Mode determines if the tool acts like a Spray, sprinkling single pixels, or a classic Airbrush, where a fine mist of pixels bleed together.

The quick key to choose the airbrush tool is A.
