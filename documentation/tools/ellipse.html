The Ellipse tool - P

This tool creates an ellipse.
If you don't activate Fill ellipse, only an outline is drawn.

Enable antialiasing to avoid jagged edges.

Holding SHIFT creates a circle, holding ALT creates the ellipse from the center instead of from "corner-to-corner".

The quick key to choose the Ellipse tool is P.
