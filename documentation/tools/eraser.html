The Eraser tool - E

You can either set the Size of your eraser or opt to Use current brush with the currently configured tip in the Brushes window.

Pressure controls how firmly the eraser touches the canvas; the lighter the touch the longer it takes until the pixels are completely erased.

You can choose how to replace the touched pixels: either with the Background color or with Transparency.

There's a difference between the Eraser and the Transparency tool: While the Transparency tool only changes the alpha values, the completely erased pixels are replaced with fully transparent black pixels (rgba: 0,0,0,0).

The quick key to choose the eraser tool is E.
