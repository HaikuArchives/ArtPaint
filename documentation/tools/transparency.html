The Transparency tool - N

This tool changes the transparency of the touched pixels.
You can either set the Size of the transparency brush or opt to Use current brush with the currently configured tip in the Brushes window.

Transparency sets the maximal percentage the touched pixels will reach.

Pressure controls how firmly the brush touches the canvas, the lighter the touch the longer it takes until the configured transparency is reached.

For example, if you set Transparency to 50 and Pressure to 100, all touched pixels will immediately turn 50% transparent (= alpha value of 128).

The quick key to choose the transparency tool is N.

The transparency tool only touches the alpha value of a pixel, not the color part (r, g, b).
What happes if you work with images with transparent areas? Well, if you increase their alpha values from 0 (= fully transparent) to something >0, the formerly invisible color part appears. Many applications use either white (255,255,255,0) or black (0,0,0,0).
If you don't want to change those transparent areas, create a selection first, quickest with Selection | All non-transparent.
