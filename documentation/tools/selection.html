The Selection tool

The Behavior determines how consecutive selections are treated. You can either Add area or Substract area.

By the way, the Edit menu of the main window holds more selection related commands: Grow / Shrink / Invert / Clear selection.

There are several Modes available:

  • Freehand lets you draw an arbitrary area.

  • Rectangle selects in the shape of a rectangle.

  • Intelligent scissors tries to select along detected edges.

  • Magic wand chooses all pixels of a similar color. You set the Tolerance for what's considerd 'similar' with the slider below.
