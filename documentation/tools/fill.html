The Fill tool - I

The fill tool applies the foreground color to the the object you left-click, and the background color if right-clicked.
The Tolerance decides if pixels on the current layer are similar enough to the color of the pixel you clicked on to be changed to the new color. A very low tolerance dyes only pixels of the exact same color. A higher tolerance will also dye shades of that color.

Flood fill only colors objects that are connected, sort of like a fill version of the magic wand selection tool.
Without Flood fill, all similar pixels on the current layer get changed, which is similar to other apps' "Replace color" tool.

You can Enable gradient and drag and drop two colors to the left and right side of the gradient bar at the bottom.
The gradient can be applied Linear, Radial, Square or Cone. Hold SHIFT to snap to 22.5Â° angles.

For Radial, Square and Cone, a click in the canvas sets the center where the gradient begins with the left color.
For Linear the left-click sets the 'flipping point' between the left and right color. If you simply clicked once without moving the mouse, you'd get a hard switch from one color to the other.
By click and dragging the mouse, you set the width and direction of the gradient.

If you Enable preview you see the gradient live as you click and move the mouse.

The quick key to choose the fill tool is I.
