The main window

This is the main window, as it appears when ArtPaint is launched or you start a new project:

You're asked for the Width and Height of the new canvas. The little widget to the right of the Width text box offers some standard dimensions. After clicking Create canvas you can start painting.

Below the canvas is a bar offering:

  • Coordinates of the mouse pointer
  • Status / Help texts for the currently selected tool, add-on, or the menu item under the mouse pointer.
  • Zoom level shows a slider when clicked to set an arbitrary magnification, or in/decrease by 25% when clicking the + / - buttons.
  • A pane with the current brush. A left-click opens the Brushes window.
  • Two color panes show the fore- and background colors. You can switch them with a right-click and change their color with a left-click which opens the Colors window.

A few tips on non-obvious features for the impatient reader:

  • To pan around the canvas, hold SPACE and move the canvas around with the mouse (works also using just the middle mouse button).

  • To zoom the canvas, just scroll with the mouse wheel (hold SHIFT to double the zoom speed).

  • Holding SHIFT has some tools snap to certain settings, like 22,5Â° angles for the Straight Line tool, or keep a 1:1 aspect ratio for the Rectangle or Ellipse and related tools to create a square or circle. ALT switches those tools to work from 'corner-to-corner' to 'center-to-corner' mode. Better read the tools pages of this guide if you don't want to miss the details.

  • Using the right-mouse button with drawing tools, paints with the background colour.

  • Holding CTRL will temporarily switch to the Color Picker to select a new the foreground color (background color with right mouse-button) and then changes back to the previously used tool.

  • ALT TAB will hide/show all but the main window. Handy if you want to paint on the whole, unobscured canvas.

  • ESC cancels pop-up windows like the rotate/translate/scale or add-on panels.
