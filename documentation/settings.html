The settings

The settings panel provides a few options, separated into four tabs:

Windows

With these checkboxes, you decide which windows to always Keep in front of, i.e. float above, the canvas.

Undo

Depending on the amount of memory available and size and number of projects and layers you work with, you can set the number of revertable actions to Off, Unlimited, or choose Adjustable and set the number of Undo steps explicitly.

Transparency

You can change the Colors of the checker pattern that's used to represent transparency, and also its Grid size.
Defaults reverts to the standard grid size and light/dark gray colors.

Miscellaneous

You can set the mouse pointer to Tool cursor and have a small icon of the active tool beside it, or — to avoid obscuring even a few pixels — set to Cross-hair cursor.

Show brush size draws the outline of the currently set brush around the mouse pointer. A sort of preview where the paint will hit the canvas.
