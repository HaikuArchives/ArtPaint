The menus

Most menus and their items are standard or don't need much explanation. Also, a short help text appears in the status bar, if you hover over an item. Let's just move quickly through them and concentrate on the not immediately obvious ones:

ArtPaint

Everything related to the application itself:
Start a New project…, open the Settings… panel, this User manual…, info About ArtPaint, and Close the current project or Quit ArtPaint altogether.

File

All the file handling:
Open/Save… image and project files, the latter are specific to ArtPaint and contain layers and meta data. For quick access, the submenus for Recent images/projects list recently opened files.

Edit

Undo/Redo the last action (the number of reversable actions can be adjusted in the settings).

You can Cut or Copy the selection of the Active layer or All layers from their submenus.

The copied selection can then be Paste as a new layer or Paste as a new project.

Finally, you can Grow/Shrink/Invert/Clear the current selection.

Layer

You can Rotate… and Translate… the current layer/selection. It'll pop up a window to enter the angle or x/y coordinates manually, or you can just rotate/move it all directly on the canvas. A right-click sets the rotation point.

You can also Flip horizontally/vertically, Change transparency… or Clear the current layer.

Add layer creates a new layer in front of the current one. More options are available from the Layers window.

Canvas

Many of the actions in this menu are similar to the ones in the Layer menu, but apply to the whole canvas, i.e. all layers: Rotate/Translate/Flip

Crop… and Scale… open a panel to enter values manually, but you can also manipulate the canvas right in the window.

Window

Zoom in/out or Set zoom level between 25% and 800%.

Set grid lets you activate an invisible grid of 2x2, 4x4 or 8x8 pixels.

Resize to fit changes the main window size to best fit the canvas.

Last is a list of all the special panels described in this manual a bit later. It's a good idea to memorize the shortcuts to them.
